gpt2-ArticleGenerator

generate article base on title and content

GitHub repository : gpt2-ArticleGenerator repo
Open API : on Ainize

How to use

1. input article title.

2. input initial content of article

3. Press the Run button and wait for about 15 seconds to get the result.
